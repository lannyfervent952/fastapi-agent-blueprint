- search across text and vector data

- support for agent tools and AI tasks

It is built for people who want to start from a ready-made base instead of assembling everything by hand

## 🧱 How the project is organized

The blueprint uses clear parts so each piece has one job:

- **Interface layer**: handles user requests
- **Domain layer**: holds app rules
- **Data layer**: talks to the database
- **AI layer**: handles agent tasks and model calls
- **Skill layer**: stores reusable agent actions

This layout helps keep the app easy to extend and easier to read later

## 🧪 Common issues

### The app does not open

- Check that the download finished
- Make sure you extracted the zip file
- Try running the app again as a normal user
- Check whether Windows blocked the file

### The browser does not show the app

- Wait a few more seconds
- Look for a local address in the app window
- Refresh the page
- Close and reopen the app

### Database connection fails

- Check the host name
- Check the user name and password
- Make sure PostgreSQL is running
- Confirm the database name is correct
- Check that the server allows connections

### Vector search does not work

- Make sure pgvector is enabled
- Check that the database has the right extension
- Confirm your records include vector data
